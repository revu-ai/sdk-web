Autocapture

Every click is captured automatically. No code wiring required.

Identity

The SDK auto-assigns a persistent userId on first visit. Call revu.identify(realId) on login to replace it, revu.reset() on logout to rotate. Both emit $identify / $reset.

Current user: (none)

Form submit

Submits emit $form_submit with field metadata only: field_names, field_types, field_count. Values are never read. Add data-revu-mask to any element to opt out of even the field names.

Outbound + download links

Links whose hostname differs from the page emit $outbound_link. Links with a download attribute (or a known file extension) emit $file_download.

Custom events

For signals autocapture cannot see (a server-side completion, an async export, a wizard step that does not change the URL), call revu.capture(name, properties?).
