REVU SDK - default install snippet

This page demonstrates the simplest install pattern from docs/install.md: one <script async> tag plus an onload handler. The bundle is loaded from http://localhost:8787/behavior (the shortest CDN URL form, resolving to the latest published version).

The snippet

<script
  async
  src="https://cdn.revu.ai/behavior"
  onload="revu.init({ apiKey: 'revu_pk_...' })"
></script>

For tag managers, inline event handlers, or strict CSPs where the onload attribute is forbidden, use the Proxy stub pattern in install.md Path 1 (alternative). The examples/vanilla/ example demonstrates that pattern alongside the rest of the SDK surface.
