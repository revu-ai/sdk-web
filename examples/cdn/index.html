REVU SDK - recommended install snippet

This page demonstrates the recommended install pattern from docs/install.md: an async <script> tag plus an inline stub queue that buffers any revu.x(...) call fired before the SDK bundle finishes loading. The bundle is loaded from http://localhost:8787/behavior (the shortest CDN URL form, resolving to the latest published version).

The snippet

<script async src="https://cdn.revu.ai/behavior"></script>
<script>
  window.revu = window.revu || new Proxy({q:[]}, {
    get: (t, m) => m in t ? t[m] : (...a) => t.q.push([m, ...a]),
  });
  revu.init({ apiKey: "revu_pk_..." });
</script>

The stub queue makes the install bulletproof: every revu.x(...) call - from tag managers, inline event handlers, cookie banners, anywhere on the page - is captured even when it fires before the SDK finishes loading. See examples/vanilla/ for a comprehensive demo of the SDK surface using this same install pattern.
